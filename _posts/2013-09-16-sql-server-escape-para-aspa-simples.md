# SQL Server escape para aspa simples

---

Bem, essa dica será mais curta que coice de porco!

Para dar escape em aspas simples basta utilizar duas aspas simples, veja o exemplo:

```sql
SELECT 'FAQ''s' AS FAQS
```

É isso ae.
